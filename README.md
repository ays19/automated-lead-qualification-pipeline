# Automated Lead Qualification Pipeline

This contains everything you need to run the automated lead qualification pipeline app locally.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GROQ_API_KEY` in `.env` to your Groq API key (and optionally customize the model with `GROQ_MODEL`).
3. Run the app:
   `npm run dev`
